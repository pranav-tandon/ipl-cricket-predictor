About Us

This web-app is created for CS 242 Final Project.

Extracted data from https://cricsheet.org/ and csv files; processed those csv files to produce clean innings wise data.

Thereby, used these clean files to generate Machine Learning model using Python APIs like scikit-learn, numpy and pandas.

Finally, used Django framework to deploy machine learning models to make predictions.
